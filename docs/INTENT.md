# Project Intent

## Why this project exists
The MET has rich but non-linear spatial logic across wings and building eras. The goal is to turn this complexity into a strength by creating personalized, interest-driven visitor routes that uncover meaningful transitions between galleries.

## Product intent
- Build a retrieval and recommendation backend around MET collection data from the website.
- Cluster and rank artworks by location, style, and narrative/story themes.
- Provide a frontend where visitors can express interests through text and keyword picks.
- Return a dynamic route that balances thematic relevance with practical movement across museum spaces.

## Design principles
- Surprise through curation: embrace hidden layers rather than flattening them.
- Movement-aware storytelling: route order should support both geographic flow and emotional arc.
- Progressive fidelity: start with website-level metadata, then enrich over time.
